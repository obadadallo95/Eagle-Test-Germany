## 4. ğŸ“ Spacing & Layout Grid

### Base Unit: 4px

All spacing uses multiples of 4px for consistency.

| Token | Size | Usage |
|-------|------|-------|
| `xxs` | 2px | Micro adjustments |
| `xs` | 4px | Inline element gaps |
| `sm` | 8px | Icon margins, small gaps |
| `md` | 12px | Default spacing |
| `lg` | 16px | Card padding, standard gaps |
| `xl` | 20px | Section spacing |
| `2xl` | 24px | Large gaps |
| `3xl` | 32px | Major sections |
| `4xl` | 40px | Screen top/bottom |

| iPhone 14 Pro | 59px | 34px | 16px |

### Layout Rules

```
Screen Padding:     16px horizontal (all devices)
Card Padding:       16px internal
Card Gap:           12px between cards
Button Gap:         12px between buttons
Section Gap:        24px between sections
Max Content Width:  100% - 32px (16px each side)
```

## 10. ğŸ”„ Theme Switching Implementation

### Save Theme Preference (Hive)

```dart
// lib/core/services/theme_service.dart

import 'package:hive/hive.dart';

enum AppThemeMode { light, dark, system }

class ThemeService {
  static const _boxName = 'settings';
  static const _themeKey = 'theme_mode';
  
  static Future<void> setThemeMode(AppThemeMode mode) async {
    final box = await Hive.openBox(_boxName);
    await box.put(_themeKey, mode.name);
  }
  
  static Future<AppThemeMode> getThemeMode() async {
    final box = await Hive.openBox(_boxName);
    final value = box.get(_themeKey, defaultValue: 'system');
    return AppThemeMode.values.firstWhere(
      (e) => e.name == value,
      orElse: () => AppThemeMode.system,
    );
  }
  
  static ThemeMode toFlutterThemeMode(AppThemeMode mode) {
    switch (mode) {
      case AppThemeMode.light:
        return ThemeMode.light;
      case AppThemeMode.dark:
        return ThemeMode.dark;
      case AppThemeMode.system:
        return ThemeMode.system;
    }
  }
}
```

### Theme Provider (Riverpod)

```dart
// lib/presentation/providers/theme_provider.dart

import 'package:flutter_riverpod/flutter_riverpod.dart';

final themeModeProvider = StateNotifierProvider<ThemeModeNotifier, AppThemeMode>((ref) {
  return ThemeModeNotifier();
});

class ThemeModeNotifier extends StateNotifier<AppThemeMode> {
  ThemeModeNotifier() : super(AppThemeMode.system) {
    _loadTheme();
  }
  
  Future<void> _loadTheme() async {
    state = await ThemeService.getThemeMode();
  }
  
  Future<void> setTheme(AppThemeMode mode) async {
    await ThemeService.setThemeMode(mode);
    state = mode;
  }
}
```

### Theme Selector Widget

```dart
// lib/presentation/widgets/theme_selector.dart

class ThemeSelector extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final currentTheme = ref.watch(themeModeProvider);
    final isDark = Theme.of(context).brightness == Brightness.dark;
    
    return Container(
      padding: const EdgeInsets.all(4),
      decoration: BoxDecoration(
        color: isDark ? AppColors.darkSurface : AppColors.lightSurface,
        borderRadius: BorderRadius.circular(12),
      ),
      child: Row(
        children: [
          _buildOption(
            context, ref,
            icon: Icons.light_mode,
            label: 'Light',
            mode: AppThemeMode.light,
            isSelected: currentTheme == AppThemeMode.light,
          ),
          _buildOption(
            context, ref,
            icon: Icons.dark_mode,
            label: 'Dark',
            mode: AppThemeMode.dark,
            isSelected: currentTheme == AppThemeMode.dark,
          ),
          _buildOption(
            context, ref,
            icon: Icons.phone_android,
            label: 'System',
            mode: AppThemeMode.system,
            isSelected: currentTheme == AppThemeMode.system,
          ),
        ],
      ),
    );
  }
  
  Widget _buildOption(
    BuildContext context,
    WidgetRef ref, {
    required IconData icon,
    required String label,
    required AppThemeMode mode,
    required bool isSelected,
  }) {
    return Expanded(
      child: GestureDetector(
        onTap: () => ref.read(themeModeProvider.notifier).setTheme(mode),
        child: AnimatedContainer(
          duration: const Duration(milliseconds: 200),
          padding: const EdgeInsets.symmetric(vertical: 12),
          decoration: BoxDecoration(
            color: isSelected ? AppColors.gold : Colors.transparent,
            borderRadius: BorderRadius.circular(8),
          ),
          child: Column(
            mainAxisSize: MainAxisSize.min,
            children: [
              Icon(
                icon,
                color: isSelected 
                    ? AppColors.darkBg 
                    : Theme.of(context).colorScheme.onSurface,
                size: 20,
              ),
              const SizedBox(height: 4),
              Text(
                label,
                style: AppTypography.bodyS.copyWith(
                  color: isSelected 
                      ? AppColors.darkBg 
                      : Theme.of(context).colorScheme.onSurface,
                  fontWeight: isSelected ? FontWeight.w600 : FontWeight.normal,
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}
```
